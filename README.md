Al entrar te aparece el menú de inicio, tienes que pulsar ENTER para entrar a la selección de niveles.

Existen 2 niveles que se selecciona con el ENTER y para escoger el nivel que quieres experimentar, se usa -> y <-.

Una vez dentro del nivel, puedes moverte con -> y <-, acumular la fuerza de salto (mantener SPACE) y saltar (soltar SPACE)

Si consigues que la llave se choque con la puerta, ganas y te lleva al menú de victoria. En caso contrario (ej: te chocas con una trampa y/o te has caído de las plataformas, te lleva al meú de derrota.
